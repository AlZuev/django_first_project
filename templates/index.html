Специалисты Friesian готовы создать любой амбициозный сайт, используя при этом самые современные инструменты, ускоряющие время разработки, но сохраняющие гибкость и качество элементов сайта. Для нас нет ограничений, мы способны мнгновенно изучить новые технологии.

Наши инструменты

HTML5

Стандартный и незаменимый язык для структуирования веб-приложений.

CSS3

Язык для описания внешнего сида документа, созданного в виде разметки.

Javascript

Невероятно быстрый язык, используется для создание интерактивности веб-приложений.

Bootstrap

Широко известный и доступный набор инструментов, HTML и CSS-шаблонов, ускоряет разработку веб-приложений.

jQuery

Библиотека JavaScript, фокусирующаяся на взаимодействии HTML и JavaScript.

Django

Фреймворк для веб-приложений на языке Python, мощный инструмент для работы с базами данных.
